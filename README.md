# AI-Based Automated UML Diagram Generator

## 📌 Description
This project is an AI-powered system that automatically generates UML diagrams from user input.  
It consists of a frontend and backend architecture.

## 🚀 Features
- Generate UML diagrams automatically
- Backend processing using Python
- Frontend interface for user interaction
- Requirement-based parsing

## 🛠️ Technologies Used
- Python
- Flask (if used)
- HTML, CSS, JavaScript
- Git & GitHub

## 📂 Project Structure
ai-uml-generator/
│
├── backend/
│   ├── main.py
│   ├── parser.py
│   └── requirements.txt
│
└── frontend/

## ▶️ How to Run
1. Navigate to backend folder
2. Install dependencies:
   pip install -r requirements.txt
3. Run the server:
   python main.py
